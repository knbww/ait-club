## üß¨ Engineering Culture & Standards

To maintain university-level quality, every member follows these protocols:

1.  **Branch Protection**: No direct pushes to `main`. All code is merged via Pull Requests with peer review from a **Lead**.
2.  **Conventional Commits**: We use structured commit messages (`feat:`, `fix:`, `docs:`, `research:`) to track progress.
3.  **Documentation First**: Every project must include a `CITATION.cff`, `LICENSE`, and comprehensive `docs/` folder.
4.  **Academic Integrity**: Strict zero-tolerance policy for plagiarism. Every research piece is backed by a `references.bib`.
